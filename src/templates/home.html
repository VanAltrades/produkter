2. To return product information about your intended item (?q=), follow the steps below.

2.a. To set a product "?q" parameter, request the API like:

{{ deployed_url }}set_produkt?q={product brand}%20{product mpn}

2.b. Or set the product identifier query parameter here:

3. Then, once the product parameter has been set:

  • navigate to {{ deployed_url }}search for structured product information JSON about the product.
  • navigate to {{ deployed_url }}resources for document links JSON about the product.
  • navigate to {{ deployed_url }}questions for popular question queries about the product.
  • navigate to {{ deployed_url }}comparisons for popular comparison queries including the product.
  • navigate to {{ deployed_url }}suggestions for popular queries about the product.
